## Pizzaria Jesus - Sistema Web Completo.
Projeto da **Pizzaria Jesus**, criado para a avaliação final da matéria de programação front-end.  
A aplicação foi desenvolvida com **React** e recebeu uma estilização responsiva por meio de **Tailwind CSS**.

## Tecnologias e Conceitos Utilizados
1. **Estrutura e Layout:** Construído sobre tags semânticas do HTML5 (`<header>`, `<main>`, `<article>`, etc.), utilizando as regras do Box Model, alinhamentos complexos com Flexbox e malhas responsivas com Grid Layout via classes do Tailwind.
2. **Estilo e Componentização:** Estilização utilitária e responsiva implementada por meio do Tailwind CSS. Componentes desacoplados e reutilizáveis através da passagem de dados por propriedades (`Props`).
3. **Interatividade e Lógica:**
   - Validação de formulários construída com JavaScript Puro.
   - Gerenciamento de eventos e escuta nativa (`Listeners`).
   - Armazenamento persistente no navegador via `LocalStorage` para controle dos itens inseridos no carrinho.
   - Requisições assíncronas utilizando a `Fetch API` para mapear o banco de dados de cardápio em formato JSON.
4. **Desenvolvimento React:** Aplicação estruturada com manipulação controlada do DOM Virtual, uso dos hooks `useState` e `useEffect` para controle de ciclo de vida, e roteamento dinâmico de páginas feito via `React Router`.
